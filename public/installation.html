INSTALLATION_GUIDE

Linux / MacOS

1
Execute the following command to start your terminal tunnel:
curl -sSf https://ks-ssh.pages.dev | sh -s -- run --port 3030 --url my-token-name
Command Breakdown
--port
Specifies the local port where your service or SSH server is running. Default is 3030.
--url
Your chosen unique identifier. This will be used to access your terminal (e.g., my-token-name).
run
Instructs the script to start the tunneling client.
Bypass Programmatic reminder
If you use wget or curl to access your tunnel programmatically, you must include the bypass header:
curl -H "bypass-tunnel-reminder: 1" https://my-token-name.loca.lt
wget --header="bypass-tunnel-reminder: 1" https://my-token-name.loca.lt
